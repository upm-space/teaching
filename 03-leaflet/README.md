# Capítulo 3: Mapas Web con Leaflet

## Objetivos

- Integrar mapas interactivos en páginas web
- Trabajar con coordenadas geográficas
- Dibujar elementos (puntos, líneas, polígonos)
- Calcular distancias y conversiones de unidades

## Ejemplos

| Archivo | Descripción |
|---------|-------------|
| [01-mapa-basico.html](./01-mapa-basico.html) | Mapa centrado en Madrid con marcador |
| [02-puntos-lineas.html](./02-puntos-lineas.html) | Botones para dibujar puntos y líneas |
| [03-distancia.html](./03-distancia.html) | Medir distancia entre dos puntos (Haversine) |
| [04-ejercicio-planificador.html](./04-ejercicio-planificador.html) | **Ejercicio**: Planificador de vuelo fotogramétrico |

## Progresión

```
01-mapa-basico
    │   Incluir Leaflet, crear mapa, añadir marcador
    ▼
02-puntos-lineas
    │   Modos de dibujo, eventos click, polylines
    ▼
03-distancia
    │   Fórmula Haversine, conversión grados ↔ metros
    ▼
04-ejercicio-planificador
        Aplicar todo: UI, cálculos, waypoints
```

## Cómo usar

Abre cualquier archivo `.html` directamente en el navegador o usa Live Server.
